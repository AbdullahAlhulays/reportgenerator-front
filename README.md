# Report Generator Frontend

React/Vite frontend for a voice-to-report workflow. Users record or upload audio, choose a language and report template, review extracted fields, then finalize DOCX/PDF reports generated by the backend.

## Backend

The frontend calls the deployed Railway backend:

```text
https://voice-report-backend-production.up.railway.app
```

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

This repo includes a GitHub Pages workflow in `.github/workflows/deploy-pages.yml`.

Pushes to `main` build `dist` and deploy it with GitHub Pages.
